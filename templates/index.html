Handwritten Digit Recognition

Note: This model is underfitted and may provide incorrect predictions.

This application demonstrates a handwritten digit recognition model trained on the MNIST dataset. Users can draw a digit on the HTML canvas, which processes the input as a 28x28 grayscale image and converts it into an intensity-based array. The model predicts the digit based on this input and displays the result.

This flask application is deployed on pythonanywhere. I build a keras model and it runs on onnx runtime. This site eill be disabled on 19 March 2025.

Features

  • Interactive Drawing Canvas: Users can draw on the web page.
  • MNIST-trained Model: The model is trained on the MNIST dataset, a standard dataset for handwritten digit recognition.
  • Single Digit numbers: Can predict single digit numbers.
  • All numbers: Enable the model to recognize and interpret multi-digit numbers (e.g., 123).
  • Calculate arithmetic operations: Users can draw for example 2+2 and the model will output 4.
